# Heartrate Monitor
This project uses Arduino Nano and AD8232 boards. 
Connect:
        GND to GND
        3.3v to 3.3v
        L0- to D11
        L0+ to D10

Connecting Electrodes:
Green to left inner wrist
Yellow to upper right thigh
Red to righ inner writst

Veiw the signal output in the serial monitor
